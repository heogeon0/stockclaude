# 딜 레이더 WebSearch 체크리스트 (6종)

> 로드 시점: stock-research 또는 base-stock 의 "딜 레이더" 섹션 작성 시.
> base-stock 과 cross-link (DRY 원칙: 본 파일이 정의처).

## 6종 필수 WebSearch 카테고리

종목 분석 직전 또는 base 작성 시 6가지 모두 탐색.

### 1. Pending M&A — 해당 기업

```
"{종목명} 인수 합병 YYYY"
"{종목명} M&A 협상"
"{종목명} 매각 발표"
```

### 2. 관계사 / 모회사 / 자회사 딜

```
"{모회사} 매각 분사"
"{종목명} 자회사 매각 인수"
"{종목명} 계열사 구조조정"
```

### 3. 경쟁사 딜

```
"{경쟁사} 인수 매각"
"{산업} 인수합병 동향 YYYY"
```

### 4. 규제 / 반독점

```
"{종목명} 공정거래위원회 조사"
"{산업} 반독점 규제"
"{종목명} 과징금"
```

### 5. 주주행동주의 / 분사 압력

```
"{종목명} 주주 행동주의"
"{종목명} 분사 압력"
"{종목명} 자사주 소각 배당"
```

### 6. 대주주 지분 변동

```
"{종목명} 대량보유 공시"
"{종목명} 최대주주 변경"
"{종목명} 임원 매도 매수"
```

## 결과 포맷

각 발견 사항마다 base.md 의 "딜 레이더" 섹션에 다음 형식 기록:

```markdown
## 🔍 딜 레이더

| 딜 제목 | 상태 | 발생일 | 밸류 임팩트 | 출처 |
|---|---|---|---|---|
| Wiz 인수 완료 (Alphabet) | **공식** | 2026-04-23 | +1~2% (단기) | TechCrunch |
| 효성중공업 미국 7,870억 변압기 수주 | **공식** | 2026-02 | +5~7% | DART |
| {경쟁사} 적대적 M&A 시도 | 루머 | 2026-04 | 부정 -5% | 매체 |

### 해당 없음 시
"현재 확인된 주요 딜 현안 없음."
```

## 상태 태그 표준

- **루머**: 미공식 보도 / 시장 추측
- **공식**: 회사 공식 발표 또는 거래소 공시
- **규제심사중**: 공정위 / SEC / EU 심사 진행
- **종결**: 거래 완료 또는 무산 확정

## 밸류 임팩트 표기

각 딜에 대해 추정 가격 영향 명시:
- 단기 (1~7일): ±X%
- 중기 (1~3개월): ±X%
- 만약 인수 시 시너지 예상치 / 분사 시 SOTP 변화

## 캐시 정책

- 딜 레이더는 base 작성 또는 30일 만기 갱신 시 매번 재실행
- 단, 같은 일자 (24시간 내) 재실행 시 캐시 재사용 가능
- 단, **stock-daily 의 review_needed 플래그 누적 3+** 시 즉시 재실행 (캐시 무효)
