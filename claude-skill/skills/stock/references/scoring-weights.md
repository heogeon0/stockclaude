# 변동성 × 재무 헬스 매트릭스 (v17 핵심)

> 로드 시점: 종목별 액션 결정 / 진입 사이즈 산정 / 손절폭 결정 시.
> **v17 핵심 변경**: 단타/스윙/중장기/모멘텀 4종 타임프레임 폐지 → 12셀 매트릭스로 단일화.

---

## 등급 매핑

### 변동성 (analyze_volatility(code).regime)

| 실현 변동성 | regime | 의미 |
|---|---|---|
| <30% | **normal** | 안정 |
| 30-50% | **high** | 단기 변동 큼 |
| 50%+ | **extreme** | 전구 종목, 손절 타이트 필수 |

### 재무 헬스 (compute_score(code).breakdown.financial — 0~100)

| 점수 | 등급 | 의미 |
|---|---|---|
| 80-100 | **A** | 흑자 우량, ROE 15%+, 부채 적정, 경고 0 |
| 60-79 | **B** | 흑자 보통, ROE 5-15%, 부채 양호 |
| 40-59 | **C** | 소폭 흑자 / 적자 진입기 / 이익질 의심 |
| <40 | **D** | 적자 지속, 부채 과다, 경고 다수 |

## 12 셀 매트릭스 — 진입 사이즈 / 피라미딩 / 손절폭

| 재무 \ 변동성 | normal (<30%) | high (30~50%) | extreme (50%+) |
|---|---|---|---|
| **A급** (흑자 우량) | **풀 / 3단 / -10%** | **풀 / 2단 / -8%** | 70% / 1단 / -6% (ATR×2) |
| **B급** (흑자 보통) | **풀 / 2단 / -8%** | 70% / 1단 / -7% | 50% / 0단 / -5% (ATR×1.5) |
| **C급** (소폭 흑자/적자 진입기) | 70% / 1단 / -7% | 50% / 1단 / -6% | 30% / 0단 / -5% (ATR×1) |
| **D급** (적자 지속) | 50% / 0단 / -6% | 30% / 0단 / -5% | **❌ 진입 비추** |

### 셀별 해석

각 셀은 다음 3가지 파라미터:

1. **진입 사이즈** — 풀 / 70% / 50% / 30% / 비추
   - 풀: 계획 자금 100% 한 번에 또는 분할 진입
   - 70~30%: 분할 진입 (1차만 진입, 추가는 트리거 충족 시)
   - 비추: 위험·보상 비율 부적합 — 진입 보류 권장

2. **피라미딩 단계** — 0~3단
   - 3단: ATR×1 / ATR×2 / ATR×3 돌파 시 추가 매수
   - 2단: ATR×1 / ATR×2
   - 1단: ATR×1.5 만 추가
   - 0단: 추가 매수 없음 (1회 진입만)

3. **손절폭** — -5%~-10%
   - 보통: 평단 기준 %
   - extreme 변동성 셀: ATR 배수 기반 (×1, ×1.5, ×2)
   - 단계 손절 (경고선 / 기준선 / 손절선) 적용 가능

## 호출 매핑

```python
volatility_regime = analyze_volatility(code).regime
financial_score = compute_score(code).breakdown.financial

# 변동성 등급
if volatility_regime == 'extreme':
    vol_tier = 'extreme'
elif volatility_regime == 'high':
    vol_tier = 'high'
else:
    vol_tier = 'normal'

# 재무 등급
if financial_score >= 80:
    fin_tier = 'A'
elif financial_score >= 60:
    fin_tier = 'B'
elif financial_score >= 40:
    fin_tier = 'C'
else:
    fin_tier = 'D'

# 셀 결정
cell = MATRIX[(fin_tier, vol_tier)]
# {'size': '70%', 'pyramiding': 1, 'stop': -6, 'stop_method': 'ATR×2'}
```

## 종합 등급 (가중총점) 매핑

`compute_score(code).total_score` (5차원 → 0-100):

| 점수 | 종합 등급 |
|---|---|
| 80+ | **Premium** |
| 60-79 | **Standard** |
| 40-59 | **Cautious** |
| <40 | **Defensive** |

### 종합 등급 vs 변동성×재무 셀

- **종합 등급** = 5차원 총합 (재무 + 산업 + 경제 + 기술 + 밸류) — 종목 전반적 매력도
- **변동성×재무 셀** = 진입·청산 액션 룰 — 실제 매매 결정

둘 다 사용:
- 종합 등급 = 진입 가치 판단 (Premium / Standard 우선)
- 변동성×재무 셀 = 사이즈 / 피라미딩 / 손절 결정

## 단타/스윙/중장기 폐지 안내 (v17)

이전 시스템:
| 타임프레임 | 재무 | 산업 | 경제 | 기술 | 밸류 |
|---|---|---|---|---|---|
| 단타 | 10% | 15% | 15% | 55% | 5% |
| 스윙 | 30% | 25% | 20% | 5% | 20% |
| 중장기 | 35% | 20% | 15% | 0% | 30% |
| 모멘텀 | 15% | 25% | 15% | 40% | 5% |

→ **폐지**. 새 룰: 모든 종목 동일 5차원 가중치 (재무 30% / 산업 25% / 경제 20% / 기술 5% / 밸류 20% — 스윙 기본값 유지) + 변동성×재무 매트릭스로 차등.

이유:
1. 종목별로 한 가지 타임프레임만 적용은 비현실 (예: 효성 = 단타+스윙+중장기 다 가능)
2. **변동성·재무 상태가 실제 액션을 결정** — 이를 매트릭스로 명시화
3. 룰 단순화 — 단일 진입 결정 프로세스
