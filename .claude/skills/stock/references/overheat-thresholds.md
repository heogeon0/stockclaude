# 과열 경고 임계값

> 로드 시점: 종목별 분석 시 과열 여부 판단.

## 과열 임계값 표

| 지표 | 임계값 | 해석 |
|---|---|---|
| **RSI** | > 80 | 극과매수, 되돌림 50%+ 가능 |
| **RSI** | > 90 | **극단적 과열** — 단기 -5~-15% 조정 65% 확률 |
| **Stoch_K** | > 95 | 7일 연속 유지 시 반전 |
| **52주 고가 이격** | > 98% | 추격 금지 — 신고가 매수 33% 승률 룰 적용 |
| **ADX** | > 40 + +DI 우세 | 추세 강하지만 고점 근접 |
| **ADX** | > 60 | 극강 추세 — 추세 종료 임박 가능 |
| **20MA 이격** | > 30% | 평균회귀 매도 시그널 |
| **20MA 이격** | > 50% | **극단적 이격** — 즉시 익절 권장 |
| **변동성 (실현)** | > 50% | extreme regime — 손절 타이트 필수 |
| **변동성 (실현)** | > 80% | **포트 위험 종목** — 사이즈 축소 권장 |
| **거래량 감소** | < 60% (20일/60일) | 모멘텀 약화 |
| **거래량 감소 + 신고가** | < 70% + 신고가 갱신 | **추세 약화 위험** |

## 복합 경고 (진짜 위험 시그널)

다음 조합 발생 시 daily 보고서 최상단 ⚠️ 표시:

1. RSI 90+ + 그랜빌 SMA20/60/120 모두 매도 + 거래량 약화
   - → "강한 매도우세 + 평균회귀 임박"
2. 52주 고가 98% + RSI 80+ + 평균회귀 매도
   - → "신고가 추격 금지"
3. 변동성 extreme + 재무 D급 + 모멘텀 D급
   - → "투기 종목 — 진입 비추"

## 과열 시 액션 권장

| 과열 강도 | 권장 행동 |
|---|---|
| 단일 임계 (예: RSI 80+) | 추가 매수 보류, 트레일링 스탑 점검 |
| 2-3개 임계 동시 | **부분 익절 권장** (1/3 ~ 1/2) |
| 4개+ 임계 동시 | **즉시 부분/전량 익절** + ⚠️ 최상단 경고 |

## W17 회고 룰 강화

회고 데이터 (`get_weekly_context.rolling_stats.rule_win_rates`) 에서:

- **신고가 돌파 매수 33% 승률** → "D+1 종가 안착 확인" 룰 강제 적용
- 즉, 신고가 매수 후 D+1 에 -3%+ 깨지면 즉시 컷
- 평균회귀 매도 60% → 거래량 동반 확인 후 진입

상세 룰: → `~/.claude/skills/stock/references/weekly-context-rules.md` 참조.

## 분할 익절 적용

W17 회고: "효성 4/23 익절 후 +₩330k 추가 상승 놓침" 사례 →

- 1차 목표 도달 시 **1/3 매도** (전량 X)
- 2차 목표 도달 시 추가 1/3
- 3차 목표 또는 트레일링 스탑까지 잔여 1/3 보유

→ 변동성×재무 매트릭스의 **피라미딩 단계 = 익절 단계** (대칭)
