# Valuation 프레임워크 룰

> 로드 시점: 재무 차원 분석 후 valuation 단계 (Reverse / Forward DCF / 확률가중 / Comps).

## 1. Reverse DCF (시장 기대치 역산)

### 입력
- 현재가 (또는 시가총액)
- 상장주식수
- 재무 요약 (`compute_financials.raw_summary`)
- WACC 가정

### 출력
- **암시 매출 CAGR** — 현 주가 정당화 위해 필요한 성장률

### WACC 가정 룰

| 시장 | 종목 규모 | WACC |
|---|---|---|
| KR | 대형주 (시총 5조+) | 9% |
| KR | 중형주 (시총 1~5조) | 10-11% |
| KR | 소형주 (시총 1조-) | 12-15% |
| US | 대형주 (시총 $50B+) | 8.5% |
| US | 중형주 ($10-50B) | 10% |
| US | 소형주 (-$10B) | 12-15% |

### 영구성장률
- 기본 2.5%
- 성장 둔화 산업 (전통 제조 / 통신) 1.5%
- AI / 신기술 3.0%

### 해석 룰
- **암시 CAGR > 산업 평균 ×2** → 과도하게 낙관 (Bear 시나리오)
- **암시 CAGR ≈ 산업 평균** → Base 시나리오 수렴
- **암시 CAGR < 산업 평균 0.5x** → 보수적 (Bull 업사이드)

## 2. Forward DCF (자체 가정)

### 시나리오 3개

| 시나리오 | 매출 CAGR | EBIT 마진 | WACC | 확률 |
|---|---|---|---|---|
| Bull | +15~20% | +14~17% | 9% | 25-35% |
| Base | +8~12% | +10~12% | 9% | 40-50% |
| Bear | +0~5% | +5~8% | 10% | 15-25% |

### 산정 결과
- 각 시나리오별 적정가
- WACC + Terminal Growth 민감도 분석

## 3. 확률가중 적정가

```python
weighted_price = (
    bull_price * bull_prob
    + base_price * base_prob
    + bear_price * bear_prob
)
```

### 확률 기본값 (Asymmetric)
- Bull 35% / Base 45% / Bear 20% (낙관)
- Bull 25% / Base 50% / Bear 25% (중립)
- Bull 15% / Base 50% / Bear 35% (보수)

→ 시장 국면 + 산업 사이클 + 종목 펀더멘털에 따라 조정

### 결과 해석
- **확률가중 vs 현재가 +20%+** → 매수 후보
- **±10%** → 중립 (관망)
- **현재가 < 확률가중 ×0.8** → 매도 후보 (이미 너무 비싸다)

## 4. Trading Comps

### 피어 그룹 선정
- 같은 산업 + 비슷한 시총 (×0.3 ~ ×3)
- 7~15개 (10개 권장)
- 글로벌 피어도 1~3개 포함 가능

### 핵심 멀티플
| 멀티플 | 적용 |
|---|---|
| **PER** | 흑자 종목 |
| **EV/EBITDA** | 자본 구조 무관 비교 |
| **PBR** | 금융주 / 자산주 |
| **PSR** | 적자 / 성장주 |
| **EV/Sales** | 적자 + 성장 종목 |

### 결과 해석
- **본 종목 백분위** (피어 중 위치)
- **프리미엄/디스카운트 %** (중간값 대비)
- **이유 분석** — 왜 그런가? (성장성 / 마진 / 리스크)

## 5. SOTP (해당 시 — 지주사/대기업)

### 적용 케이스
- 지주사 (LG, SK, 삼성물산)
- 다각화 대기업 (현대차, 한화, 두산)
- 사업부 분리 분석 가능 시

### 산정 룰
- 사업부별 EV 분리 산정
- 각 사업부 적정 멀티플 적용
- 핵심 vs 비핵심 구분
- 옵션가치 (자회사 IPO 등) 별도

### 출력
- 코어 EV + 비코어 EV + 옵션가치 + 순부채 = 적정 시가총액
- 적정 주당 가격
- 현재 주가 vs 적정가 ±%
