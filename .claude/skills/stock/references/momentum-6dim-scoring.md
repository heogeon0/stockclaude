## 차원별 산정 룰

### 1. 12-1 수익률 (25점)
- 12개월 수익률에서 최근 1개월 제외 (mean reversion 효과 회피)
- 25점 만점 = +50%+
- 0점 = -50%-

### 2. 52주 고가 근접도 (20점)
- 현재가 / 52주 고가 비율
- 20점 만점 = 95%+ (신고가 근접)
- 0점 = 50%- (저점 회복 부족)

### 3. ADX 추세 강도 (15점)
- ADX(14) — 추세 강도 지표
- 15점 = 30+
- 0점 = 15-

### 4. Sharpe 조정 수익률 (15점)
- (수익률 - 무위험) / 변동성
- 15점 = 1.5+
- 0점 = 0-

### 5. MA 정렬 점수 (15점)
- 5/20/60/120/200 일선 정배열 강도
- 15점 = 완전 정배열 (현재가 > 5MA > 20MA > 60MA > 120MA > 200MA)
- 0점 = 역배열

### 6. 거래량 동반 (10점)
- 20일 평균 거래량 / 60일 평균
- 10점 = 1.5x+
- 0점 = 0.5x-

## 등급 해석

| 점수 | 등급 | 의미 |
|---|---|---|
| 80+ | **A+** | Top 10% — 모멘텀 강함 |
| 65-79 | **A** | 견조 — 유지/진입 가능 |
| 50-64 | **B** | 중립 — 추가 확인 |
| 35-49 | **C** | 약함 — 매수 대기 |
| <35 | **D** | 부재 — 회피 |

## Z-score 상대 랭킹

`rank_momentum(codes, market='kr')` 결과:
- 각 종목의 **점수**를 평균 / 표준편차로 정규화
- Z-score +1.0+ = 상위 16%
- Z-score +2.0+ = 상위 2.5%

→ 절대 점수 + 상대 Z-score 둘 다 고려

## 모멘텀 피로도

종목 추세 지속 일수 추적:

| 일수 | 단계 | 의미 |
|---|---|---|
| 0-30일 | 초기 | 진입 적합 |
| 30-90일 | 중기 | 진입 가능 (단, 이격 큰 상태) |
| 90-180일 | 후기 | 신규 진입 신중 (피크 가능) |
| 180일+ | 피크 | **신규 진입 중단** — 거래량 감소 동반 시 청산 검토 |

## 광역 스크리닝 (rank_momentum_wide)

전체 시장 시총 상위 30~50 중 모멘텀 Top N 추출:

```
rank_momentum_wide(
  market='kr',
  top_n=30,
  min_market_cap_krw=3_000_000_000_000,  # 3조+
)
```

→ Top 30 + 섹터 분포 분석 + Dual Momentum 시그널 부착

## 신규 발굴 시 활용

`stock-discover` 의 1단계 광역 스크리닝에서 사용:
- 유동성 (거래대금 상위 50) ∩ 모멘텀 (상위 30) → 후보군 30~50개
- 후보 각각에 대해 6차원 정량 분석 (research) → Top 3~5 선별

## 보유 종목 모멘텀 추적

`stock-daily` 의 종목별 분석에서:
- `rank_momentum(holdings)` 호출 → 보유 종목 상호 모멘텀 비교
- 모멘텀 점수 50 미만 + 순위 30 밖 → 청산 후보
